# Heal.in
Kecemasan dan gangguan mental pada milenial dan Gen Z meningkat akibat burnout, tekanan ekonomi, FOMO, dan ketidakpastian masa depan. Akses psikolog mahal dan terpusat di kota besar. Aplikasi yang ada kurang menjamin privasi. Dibutuhkan solusi terjangkau, anonim, berbasis komunitas, dan aman secara teknis.
